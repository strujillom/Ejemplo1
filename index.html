Fundamentos de las Tecnologías Web

HTML: La estructura

HTML (HyperText Markup Language) es el lenguaje utilizado para estructurar el contenido de una página web. Define la semántica y la organización de los elementos en la página.

  • Elementos: Son las unidades básicas de construcción en HTML. Ejemplos: <h1> (encabezado), <p> (párrafo), <div> (contenedor).
  • Atributos: Proporcionan información adicional sobre los elementos. Ejemplos: id, class, href.

CSS: El estilo

CSS (Cascading Style Sheets) es el lenguaje utilizado para dar estilo a las páginas web. Define cómo se muestran los elementos HTML en la pantalla.

  • Selectores: Se utilizan para seleccionar los elementos HTML a los que se aplicarán los estilos. Ejemplos: elementos por nombre de etiqueta, por id, por clase.
  • Propiedades: Definen las características visuales de los elementos. Ejemplos: color, font-size, background-color.
  • Valores: Especifican el valor de una propiedad. Ejemplos: red, 16px, #FFFFFF.

JavaScript: La interactividad

JavaScript es un lenguaje de programación que se utiliza para agregar interactividad a las páginas web. Permite crear aplicaciones web dinámicas y complejas.

  • Variables: Almacenan datos.
  • Funciones: Realizan tareas específicas.
  • Eventos: Responden a acciones del usuario (clics, movimientos del ratón).
